Bubble Sort

Bubble Sort was initially referred to as "Exchange Sort" and is now referred to as "Sinking Sort".

The term "Bubble Sort" was first used by Iverson in 1962.

Bubble sort is a sorting algorithm that sorts an array by comparing an element to its neighboring element. After the comparison, the elements are swapped if they are not in chronological order.

Bubble sort is a comparison-based sorting algorithm where the comparison of adjacent elements is primitive act.

On each pass, it compares adjacent elements in the array and swaps those that are not in sorted manner.

Basically, each pass through the array places the next largest value in its proper place, i.e. the comparison count is decremented by one on each pass.

In other words, each item "bubbles" into place where it should be in sorted order.

This invariant is maintained by the algorithm in each pass and therefore bubble sort correctly outputs the sorted sequence.


Time Complexity

Best Case: O(1)
Worst Case: O(n2)
Average case: O(n2)


Space Complexity

Auxiliary Space: O(1)
